什么是损失函数？

损失函数（Loss Function）是一个单一数字，用来衡量模型预测值与真实值之间的差距。训练神经网络的目标就是最小化损失函数。

类比：高尔夫分数

损失就像高尔夫的分数——越低越好。你的每一次挥杆（训练迭代）都试图让球更接近球洞（真实值），最终目标是以最少的杆数完成比赛。

没有损失函数，模型就不知道自己学得好不好、该往哪个方向调整。它是连接"预测"和"学习"的桥梁。

均方误差 (MSE)

MSE 是回归任务中最常用的损失函数，计算预测值与真实值之差的平方的平均值。

$$L_{MSE} = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2$$

  • 平方操作让大误差被放大惩罚（对异常值敏感）
  • 处处可微，优化方便
  • 单位是目标变量的平方（量纲不直观）

在上面的交互实验中，你可以看到 MSE 对偏离较远的点惩罚更大——误差被平方后，离得越远的点贡献的损失越多。

平均绝对误差 (MAE)

MAE 计算预测值与真实值之差的绝对值的平均值。

$$L_{MAE} = \frac{1}{n}\sum_{i=1}^{n}|y_i - \hat{y}_i|$$

  • 对所有误差一视同仁（不放大异常值）
  • 比 MSE 更鲁棒
  • 在零点处不可微（实践中影响不大）

在交互实验中切换 MSE 和 MAE，观察损失值的变化——MSE 对离群点更敏感。

交叉熵损失 (Cross-Entropy)

交叉熵是分类任务的标准损失函数，衡量两个概率分布之间的差异。

二元交叉熵（Binary Cross-Entropy）

用于二分类问题，标签 $y_i \in \{0, 1\}$，预测概率 $\hat{y}_i \in (0, 1)$：

$$L_{BCE} = -\frac{1}{n}\sum_{i=1}^{n}\left[y_i\log(\hat{y}_i) + (1-y_i)\log(1-\hat{y}_i)\right]$$

多类交叉熵（Categorical Cross-Entropy）

用于多分类问题，$K$ 个类别，$y_{i,k}$ 为 one-hot 标签，$\hat{y}_{i,k}$ 为 softmax 输出概率：

$$L_{CE} = -\frac{1}{n}\sum_{i=1}^{n}\sum_{k=1}^{K} y_{i,k}\log(\hat{y}_{i,k})$$

为什么分类不用 MSE？

对于分类任务，MSE 的梯度在预测概率接近 0 或 1 时会非常小，导致学习缓慢。而交叉熵的梯度形式更简洁（$\hat{y} - y$），训练更高效。交叉熵源于信息论——它衡量用预测分布编码真实标签所需的额外信息量。

如何选择损失函数？

任务类型 输出激活 损失函数 说明
回归 Linear MSE / MAE MSE 更常用；若有异常值，MAE 更鲁棒
二分类 Sigmoid Binary Cross-Entropy 输出为正类概率
多分类 Softmax Categorical Cross-Entropy 输出为各类别概率分布
图像分割 Sigmoid/Softmax Dice Loss / CE 后续 UNet 章节会详述

损失曲面（Loss Landscape）

如果把模型的所有参数看作坐标轴，损失函数在每个参数组合下的值就构成了一个高维曲面。训练的过程就是在这个曲面上找到最低点（全局最小值或局部最小值）。

损失曲面的形状对训练难度有很大影响——平坦的区域学习慢，陡峭的区域可能出现梯度爆炸。下一章我们将深入探讨如何在损失曲面上高效地"下山"。
