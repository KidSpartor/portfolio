第 2 章 · 感知机：最简单的神经网络

感知机（Perceptron）是所有神经网络的起点。它简单、直观，却是理解深度学习的基石。

交互式感知机模拟器

$$\hat{y} = \text{step}(w_1 x_1 + w_2 x_2 + b)$$
正确: 0 / 错误: 0 / 总计: 0
Epoch: 0

直觉理解

🎯 投票类比：想象一个评审委员会。每位评委（输入）根据自己的重要性（权重）投票。如果总票数超过某个门槛（偏置），就通过（输出 1）；否则拒绝（输出 0）。感知机就是这样一个简单的"多数决"系统。

核心组件

  • 权重 (Weights) — 每个输入的重要性。$w_1$ 和 $w_2$ 控制各输入对最终决策的影响程度。
  • 偏置 (Bias) — 决策的门槛。相当于"通过的最低分数线"。
  • 阶跃函数 (Step Function) — 如果加权求和 > 0，输出 1；否则输出 0。

数学公式

$$z = w_1 x_1 + w_2 x_2 + b$$ $$\hat{y} = \text{step}(z) = \begin{cases} 1 & \text{if } z > 0 \\ 0 & \text{otherwise} \end{cases}$$

训练规则

感知机的学习规则非常简单：

$$w_i \leftarrow w_i + \eta \cdot (y - \hat{y}) \cdot x_i$$ $$b \leftarrow b + \eta \cdot (y - \hat{y})$$

其中 $\eta$ 是学习率，$y$ 是真实标签，$\hat{y}$ 是预测值

预测正确时 ($y = \hat{y}$)，权重不变；预测错误时，权重朝正确方向调整。

代码实现

import numpy as np

class Perceptron:
    def __init__(self, lr=0.1):
        self.w = np.random.randn(2)  # 两个权重
        self.b = 0.0                    # 偏置
        self.lr = lr                     # 学习率

    def predict(self, x):
        z = np.dot(self.w, x) + self.b
        return 1 if z > 0 else 0

    def train(self, X, y, epochs=100):
        for _ in range(epochs):
            for xi, yi in zip(X, y):
                y_hat = self.predict(xi)
                error = yi - y_hat
                self.w += self.lr * error * xi
                self.b += self.lr * error

# AND 门示例
X = np.array([[0,0], [0,1], [1,0], [1,1]])
y = np.array([0, 0, 0, 1])

p = Perceptron(lr=0.1)
p.train(X, y)
print("训练后预测：", [p.predict(xi) for xi in X])
# 输出: [0, 0, 0, 1] ✓

局限性：XOR 问题

感知机只能解决线性可分的问题。1969 年，Minsky 和 Papert 证明了感知机无法解决异或（XOR）问题，这一发现曾让神经网络研究陷入低潮。

为什么 XOR 不可分？

在上图中，蓝色的 $(0,0)$ 和 $(1,1)$ 属于类 0，红色的 $(0,1)$ 和 $(1,0)$ 属于类 1。无论你如何画一条直线，都无法将红色和蓝色完全分开。这就是线性不可分的含义。
