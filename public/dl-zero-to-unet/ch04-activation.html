第 4 章 · 激活函数

让神经网络拥有非线性表达能力的关键

交互实验：激活函数可视化

拖拽曲线上的点来查看具体数值

为什么需要非线性？

如果神经网络每一层都只做线性变换，那么无论堆叠多少层，最终结果等价于一个单独的线性变换。

证明：堆叠线性层 = 单层线性

设第一层输出 $h = W_1 x + b_1$，第二层输出 $y = W_2 h + b_2$，代入得：

$$y = W_2(W_1 x + b_1) + b_2 = (W_2 W_1)x + (W_2 b_1 + b_2) = W'x + b'$$

其中 $W' = W_2 W_1$，$b' = W_2 b_1 + b_2$。无论叠多少层，始终是一个线性映射，无法学习 XOR 等非线性问题。

因此，我们需要在层与层之间引入非线性激活函数，打破这种等价性，赋予网络逼近任意函数的能力。

Sigmoid

Sigmoid 函数将任意实数压缩到 $(0, 1)$ 区间，形状像一个 "S"。

$$\sigma(x) = \frac{1}{1 + e^{-x}}$$

导数：$\sigma'(x) = \sigma(x)(1 - \sigma(x))$

  • 输出范围：$(0, 1)$，可解释为概率
  • 优点：平滑、可微，输出有界
  • 缺点：梯度最大值仅 $0.25$（在 $x=0$ 处），当 $|x|$ 较大时梯度接近 $0$，导致梯度消失

梯度消失问题

Sigmoid 的导数最大只有 0.25。当网络层数较深时，梯度经过多次乘积后会指数级衰减，导致前面的层几乎无法学习。这是深度网络早期面临的核心困难之一。

Tanh

Tanh（双曲正切）是 Sigmoid 的"升级版"，输出以零为中心。

$$\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$$

导数：$\tanh'(x) = 1 - \tanh^2(x)$

  • 输出范围：$(-1, 1)$，零中心化
  • 优点：零中心使得后续层的输入分布更均匀，有助于训练
  • 缺点：同样存在梯度消失问题（导数最大值为 $1$，但在 $|x|$ 较大时迅速衰减）

ReLU

ReLU（Rectified Linear Unit）是目前最常用的激活函数，简单而高效。

$$\text{ReLU}(x) = \max(0, x)$$

导数：$$\text{ReLU}'(x) = \begin{cases} 1 & x > 0 \\ 0 & x \le 0 \end{cases}$$

  • 输出范围：$[0, +\infty)$
  • 优点：计算简单、梯度不会消失（正区间梯度恒为 1）、稀疏激活
  • 缺点：当输入为负时梯度为 0，神经元"死亡"（Dead ReLU 问题）

为什么 ReLU 是默认选择？

ReLU 的正区间梯度恒为 1，不会像 Sigmoid/Tanh 那样梯度消失。这让深度网络的训练变得可行。在实践中，大多数隐藏层都首选 ReLU。

Leaky ReLU

为了解决 Dead ReLU 问题，Leaky ReLU 在负区间给一个小的斜率。

$$\text{LeakyReLU}(x) = \begin{cases} x & x > 0 \\ \alpha x & x \le 0 \end{cases}$$

其中 $\alpha$ 通常取 $0.01$。导数在负区间为 $\alpha$（小正数），避免了梯度完全为零。

激活函数对比

激活函数 公式 输出范围 零中心 梯度消失 适用场景
Sigmoid $\frac{1}{1+e^{-x}}$ $(0, 1)$ 否 严重 二分类输出层
Tanh $\frac{e^x-e^{-x}}{e^x+e^{-x}}$ $(-1, 1)$ 是 中等 RNN、归一化层
ReLU $\max(0,x)$ $[0, +\infty)$ 否 无（正区间） 隐藏层（默认）
Leaky ReLU $\max(\alpha x, x)$ $(-\infty, +\infty)$ 否 无 替代 ReLU
Linear $x$ $(-\infty, +\infty)$ 是 无 回归输出层

什么时候用什么？

隐藏层

首选 ReLU，简单高效。如果遇到 Dead ReLU 问题，可尝试 Leaky ReLU。

二分类输出

使用 Sigmoid，输出可解释为属于正类的概率 $P(y=1|x) \in (0,1)$。

多分类输出

使用 Softmax，将 logits 转化为概率分布，各类别概率之和为 1。

回归输出

使用 Linear（无激活），让输出可以取任意实数值。
