第 11 章 · 动手实践：从零搭建 UNet

恭喜你来到最后一章！在前面的章节中，你已经学习了深度学习的核心概念。现在，让我们把所有知识融合在一起，用 PyTorch 从零开始搭建一个完整的 UNet 模型。

一、构建模块

UNet 由几个重复使用的构建块组成。让我们从最基础的双卷积块开始。

1.1 双卷积块（Double Convolution）

每个编码器/解码器层都包含两个连续的 3×3 卷积 + BatchNorm + ReLU：

unet.py Python 
# 双卷积块
class DoubleConv(nn.Module):
    def __init__(self, in_ch, out_ch):
        super().__init__()
        self.conv = nn.Sequential(
            nn.Conv2d(in_ch, out_ch, 3, padding=1),
            nn.BatchNorm2d(out_ch),
            nn.ReLU(inplace=True),
            nn.Conv2d(out_ch, out_ch, 3, padding=1),
            nn.BatchNorm2d(out_ch),
            nn.ReLU(inplace=True)
        )

    def forward(self, x):
        return self.conv(x)

1.2 编码器块（Encoder Block）

编码器块 = 双卷积 + 最大池化。返回池化后的特征和卷积输出（供跳跃连接使用）：

unet.py Python 
class EncoderBlock(nn.Module):
    def __init__(self, in_ch, out_ch):
        super().__init__()
        self.conv = DoubleConv(in_ch, out_ch)
        self.pool = nn.MaxPool2d(2)

    def forward(self, x):
        # 返回卷积输出（跳跃连接用）和池化输出
        skip = self.conv(x)
        x = self.pool(skip)
        return skip, x

1.3 解码器块（Decoder Block）

解码器块 = 上卷积 + 拼接跳跃连接 + 双卷积：

unet.py Python 
class DecoderBlock(nn.Module):
    def __init__(self, in_ch, out_ch):
        super().__init__()
        self.up = nn.ConvTranspose2d(in_ch, out_ch, 2, stride=2)
        self.conv = DoubleConv(in_ch, out_ch)  # in_ch 因为拼接后翻倍

    def forward(self, x, skip):
        x = self.up(x)
        # 拼接跳跃连接（处理尺寸差异）
        x = torch.cat([x, skip], dim=1)
        x = self.conv(x)
        return x

1.4 完整 UNet 模型

将所有模块组装成完整的 UNet：

unet.py Python 
class UNet(nn.Module):
    def __init__(self, in_ch=1, out_ch=1):
        super().__init__()
        # 编码器
        self.enc1 = EncoderBlock(in_ch, 64)
        self.enc2 = EncoderBlock(64, 128)
        self.enc3 = EncoderBlock(128, 256)
        self.enc4 = EncoderBlock(256, 512)
        # 瓶颈层
        self.bottleneck = DoubleConv(512, 1024)
        # 解码器
        self.dec4 = DecoderBlock(1024, 512)
        self.dec3 = DecoderBlock(512, 256)
        self.dec2 = DecoderBlock(256, 128)
        self.dec1 = DecoderBlock(128, 64)
        # 输出层
        self.out_conv = nn.Conv2d(64, out_ch, 1)

    def forward(self, x):
        # 编码
        s1, x = self.enc1(x)
        s2, x = self.enc2(x)
        s3, x = self.enc3(x)
        s4, x = self.enc4(x)
        # 瓶颈
        x = self.bottleneck(x)
        # 解码（使用跳跃连接）
        x = self.dec4(x, s4)
        x = self.dec3(x, s3)
        x = self.dec2(x, s2)
        x = self.dec1(x, s1)
        # 输出
        return self.out_conv(x)

二、数据准备

医学图像分割需要成对的 图像 和 掩码（mask）。以下代码展示如何加载数据并进行数据增强：

dataset.py Python 
import torch
from torch.utils.data import Dataset
from torchvision import transforms
from PIL import Image
import os

class MedicalDataset(Dataset):
    def __init__(self, img_dir, mask_dir, size=256):
        self.img_paths = sorted([
            os.path.join(img_dir, f) for f in os.listdir(img_dir)
        ])
        self.mask_paths = sorted([
            os.path.join(mask_dir, f) for f in os.listdir(mask_dir)
        ])
        self.size = size
        # 图像增强
        self.img_transform = transforms.Compose([
            transforms.Resize((size, size)),
            transforms.RandomHorizontalFlip(),
            transforms.RandomVerticalFlip(),
            transforms.RandomRotation(15),
            transforms.ToTensor(),
            transforms.Normalize([0.5], [0.5])
        ])
        self.mask_transform = transforms.Compose([
            transforms.Resize((size, size)),
            transforms.RandomHorizontalFlip(),
            transforms.RandomVerticalFlip(),
            transforms.RandomRotation(15),
            transforms.ToTensor()
        ])

    def __len__(self):
        return len(self.img_paths)

    def __getitem__(self, idx):
        img = Image.open(self.img_paths[idx]).convert('L')
        mask = Image.open(self.mask_paths[idx]).convert('L')
        # 使用相同的随机种子确保图像和掩码变换一致
        seed = torch.randint(0, 2**32, (1,)).item()
        torch.manual_seed(seed)
        img = self.img_transform(img)
        torch.manual_seed(seed)
        mask = self.mask_transform(mask)
        return img, mask

三、训练过程

训练 UNet 使用 Binary Cross-Entropy Loss（二值交叉熵损失）或 Dice Loss，配合 Adam 优化器：

train.py Python 
import torch.optim as optim

# 初始化模型、损失函数、优化器
model = UNet(in_ch=1, out_ch=1).cuda()
criterion = nn.BCEWithLogitsLoss()
optimizer = optim.Adam(model.parameters(), lr=1e-4)

# 训练循环
num_epochs = 50
for epoch in range(num_epochs):
    model.train()
    total_loss = 0

    for images, masks in train_loader:
        images = images.cuda()
        masks = masks.cuda()

        # 前向传播
        outputs = model(images)
        loss = criterion(outputs, masks)

        # 反向传播
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()

        total_loss += loss.item()

    avg_loss = total_loss / len(train_loader)

四、评估指标

图像分割中最常用的两个评估指标是 IoU（交并比）和 Dice 系数。

IoU (Intersection over Union)
$$\text{IoU} = \frac{|A \cap B|}{|A \cup B|}$$

预测区域与真实区域的交集除以并集

Dice 系数
$$\text{Dice} = \frac{2|A \cap B|}{|A| + |B|}$$

两倍交集除以两区域面积之和，等价于 F1 分数

metrics.py Python 
def compute_iou(pred, target):
    """计算 IoU (Jaccard Index)"""
    intersection = (pred & target).sum()
    union = (pred | target).sum()
    return intersection.float() / (union.float() + 1e-8)

def compute_dice(pred, target):
    """计算 Dice 系数"""
    intersection = (pred & target).sum()
    return (2 * intersection.float()) / (
        pred.sum().float() + target.sum().float() + 1e-8
    )

evaluate.py Python 
def visualize_prediction(image, mask, pred):
    """可视化预测结果"""
    # 创建 RGB 重叠图
    overlay = torch.zeros(3, *mask.shape)

    # 真阳性: 预测和真实都是白色 -> 绿色
    tp = (pred == 1) & (mask == 1)
    overlay[1][tp] = 1.0  # Green

    # 假阳性: 预测白色但真实黑色 -> 红色
    fp = (pred == 1) & (mask == 0)
    overlay[0][fp] = 1.0  # Red

    # 假阴性: 预测黑色但真实白色 -> 蓝色
    fn = (pred == 0) & (mask == 1)
    overlay[2][fn] = 1.0  # Blue

    return overlay
